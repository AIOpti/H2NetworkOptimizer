Key Concept: Merit Order

Cheapest producers run first. By-products ($0.008) before SMRs ($0.035) before electrolysers ($0.065). Import ($0.075) is last resort.

Key Concept: MIP

Mixed Integer Programming finds the mathematically optimal dispatch — not just a good guess. Uses binary ON/OFF variables plus continuous flow rates.

Key Concept: Purifiers

PSA (85% recovery, 99.9% purity) and MEM (92% recovery, 98% purity) clean hydrogen but lose some in the process. This is why you may need import despite having surplus production.
